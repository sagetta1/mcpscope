mcpscope

DevTools for the Model Context Protocol — passive capture of every JSON-RPC message between Claude Desktop / Cursor / Continue and your MCP server.

What it does

  • Sits in the path. One mcpscope install wraps every MCP server in your Claude Desktop or Claude Code config. No code changes, no SDK. Restart Claude — every JSON-RPC request and response is captured to a local SQLite DB.
  • Persistent timeline you can grep, diff, replay. Browse sessions in a local web UI on localhost:3939. Side-by-side request/response, structured diff between two calls of the same tool, live SSE feed for the session running right now.
  • Local-first, single binary, Apache 2.0. One Go binary, ~15 MB. SQLite is the only datastore. No cloud, no telemetry, no account. Token cost overlay (cl100k_base, ±20% for Claude). Source is github.com/sagetta1/mcpscope.

Try it from source today

git clone https://github.com/sagetta1/mcpscope
cd mcpscope && make build
./mcpscope install                  # dry-run, shows the diff
./mcpscope install --apply          # backup + atomic write
./mcpscope ui                       # opens http://localhost:3939
